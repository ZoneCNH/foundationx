# 发布说明（Release Notes）

## 版本 v0.1.0

`foundationx` L0 contract module 的初始 release。

包含：

- Error、health、lifecycle、retry、sanitizer、clock 与 version contracts。
- Focused unit tests 与 examples。
- JSON contract schemas。
- CI、security 与 release workflows。
- Release manifest generation。

已知约束：

- Standard-library-only dependency policy。
- 无 concrete infrastructure adapters。
- 尚未生成 Go API documentation site。
